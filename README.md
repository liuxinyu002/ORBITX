# OrbitX

**隐私优先、完全离线的桌面 AI 效率工具箱。**

OrbitX 是一款基于 Tauri 构建的 AI 原生桌面工具箱，面向知识工作者（招聘、研究、数据分析等场景）。首个工具「结构化提取器」通过全局快捷键和系统无障碍 API，在不打断工作流的前提下，将碎片化信息一键提取为结构化数据，并持久化到本地数据库。更多工具将持续加入。无账号、无云端、无遥测——数据完全归属于你。

**技术栈：** Tauri v2 + React 18 + TypeScript + Rust + SQLite + Tailwind CSS

**核心原则：**
- **隐私优先** — 所有数据存储在本地 SQLite，不经过任何服务器
- **快捷键驱动** — 全局热键触发，不离开当前应用即可完成提取
- **克制设计** — 系统原生字体，单色系配色，无渐变无动画装饰

更多产品理念见 [PRODUCT.md](./PRODUCT.md)，设计系统见 [DESIGN.md](./DESIGN.md)。

## 工具

### 结构化提取器

从任意应用中选中文本，通过 AI 按自定义字段提取结构化数据，存入本地数据库，支持浏览和导出。

**典型场景：**
- 从招聘网站批量提取候选人简历信息
- 从电商页面采集商品规格和价格
- 从论文/报告中整理研究数据

#### 快速上手

1. **配置 AI 模型** — 进入设置页，填入 API Key（支持 DeepSeek、智谱、OpenAI 及 Ollama 本地模型）
2. **创建提取任务** — 在「结构化提取器」页面新建任务，手动添加字段或通过自然语言描述让 AI 自动生成 Schema
3. **激活任务** — 点击任务旁的开关，将其设为当前激活任务
4. **选中文本，按快捷键** — 在任意应用中选中目标文本，按 `Cmd/Ctrl+Shift+E` 直接提取，或按 `Cmd/Ctrl+Shift+K` 唤出命令面板选择目标任务
5. **查看结果** — 切回 OrbitX 的「数据浏览」标签页，查看提取结果，支持 CSV/XLSX 导出

#### 核心操作

| 操作 | 方式 |
|------|------|
| 静默提取 | `Cmd/Ctrl+Shift+E`（抓取当前应用选中文本，按激活任务提取） |
| 命令面板 | `Cmd/Ctrl+Shift+K`（唤出居中面板，选择目标任务进行提取，同时将该任务设为新的激活任务） |
| 创建/编辑字段 | 提取器页面 → 字段配置 Tab → 手动添加或 AI 生成 |
| 浏览数据 | 提取器页面 → 数据浏览 Tab → 分页表格 |
| 导出数据 | 数据浏览页面 → 导出按钮 → 选择 CSV 或 XLSX |
| 删除记录 | 数据浏览页面 → 点击删除 → 再次确认 |

> 如果 AI 判定选中文本与任务不相关，会自动弹出预览窗口，允许你手动选择任务或强制入库。
